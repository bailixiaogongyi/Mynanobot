# 字体大小调整计划

## 问题分析

项目中存在小于 12px 的字体大小，需要统一调整到最小 12px。

### 当前字体大小分布

| rem 值     | px 值 | 数量  | 状态    |
| --------- | ---- | --- | ----- |
| 0.625rem  | 10px | 4 处 | ❌ 需修改 |
| 0.6875rem | 11px | 4 处 | ❌ 需修改 |
| 0.75rem   | 12px | 多处  | 同步增大  |
| 0.8125rem | 13px | 多处  | 同步增大  |
| 0.875rem  | 14px | 多处  | 同步增大  |
| 0.9375rem | 15px | 多处  | 同步增大  |
| 11px      | 11px | 1 处 | ❌ 需修改 |

### 需要修改的文件

由于需要保持整体协调，最小字体调整后，其他字体应该同步增加，保证页面展示主次的分明和层次感。

## 实施方案

### 方案：批量替换

需要将所有字体同步增加，具体调整策略如下：

1. `0.625rem` → `0.75rem`
2. `0.6875rem` → `0.78rem`
3. `0.75rem` → `0.8rem`
4. `0.8125rem` → `0.85rem`
5. `0.875rem` → `0.92rem`
6. `0.9375rem` → `0.98rem`
7. `11px` → `0.78rem`

<br />

## 执行步骤

1. 梳理所有需要修改的文件：
2. 逐个文件进行字体大小的修改
3. 验证字体大小修改的情况
4. 回复修改的整体情况
