# 临时笔记技能

## 功能

- 在 `pending/` 目录下创建临时笔记文件
- 记录临时想法、琐碎信息
- 作为临时缓存，定期清理或转移到其他笔记

## 目录位置

```
workspace/
└── pending/           # 临时笔记目录
    ├── 临时事项.md
    ├── 想法记录.md
    └── 琐事速记.md
```

## 使用方式

### 创建临时笔记

当用户明确要求创建临时笔记时，使用 `write_file` 工具在 `pending/` 目录下创建文件。

示例：

```
write_file("pending/临时事项.md", "# 临时事项\n\n- [ ] 任务1\n- [ ] 任务2\n")
```

### 追加内容

使用 `edit_file` 工具追加内容到临时笔记。

示例：

```
edit_file("pending/临时事项.md", old_content="- [ ] 任务2\n", new_content="- [ ] 任务2\n- [ ] 新任务\n")
```

### 查看临时笔记

使用 `read_file` 工具读取临时笔记。

示例：

```
read_file("pending/临时事项.md")
```

### 列出所有临时笔记

使用 `list_dir` 工具列出 `pending/` 目录。

示例：

```
list_dir("pending")
```

### 清理临时笔记

1. 读取临时笔记内容
2. 将有价值的内容转移到：
   - 每日笔记：当天的事件
   - 主题笔记：长期积累的内容
   - 项目笔记：项目相关信息
3. 删除已处理的临时笔记

```
delete_file("pending/临时事项.md")
```

## 使用场景

| 场景                 | 操作                                  |
| -------------------- | ------------------------------------- |
| 用户要求记录临时想法 | 在 `pending/` 下创建或追加笔记        |
| 用户要求记录待办事项 | 创建或更新 `pending/临时事项.md`      |
| 琐事速记             | 创建 `pending/琐事速记-YYYY-MM-DD.md` |
| 临时收集信息         | 创建主题相关的临时文件                |

## 与其他笔记的区别

| 类型     | 用途               | 位置                   |
| -------- | ------------------ | ---------------------- |
| 每日笔记 | 记录当天事件、任务 | `memory/YYYY-MM-DD.md` |
| 临时笔记 | 快速记录临时想法   | `pending/{主题}.md`    |
| 主题笔记 | 深度记录特定主题   | `topics/{主题}.md`     |
| 项目笔记 | 项目相关信息       | `projects/{项目名}.md` |

## 注意事项

- 临时笔记不适合存储重要信息
- 定期清理，避免积累过多内容
- 处理完成后及时删除或转移到其他笔记
- 文件命名应清晰表达内容主题
