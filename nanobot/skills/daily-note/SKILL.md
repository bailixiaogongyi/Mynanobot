# 每日笔记技能

## 功能

- 自动在 `memory/` 目录下创建每日笔记文件 `YYYY-MM-DD.md`
- 自动继承：创建新笔记时从前一天获取"今日待办"和"长期待办"
- 自动清理：前一天的长期待办在继承后标记为已继承（不再重复）
- 支持追加内容到当日笔记
- 支持查看最近几天的笔记
- 超过 7 天的笔记会被归档到 `daily/` 目录

## 目录位置

```
workspace/
└── memory/            # 每日笔记目录（最近 7 天）
    ├── MEMORY.md      # 长期记忆（无待办）
    ├── HISTORY.md     # 历史日志
    ├── 2026-03-27.md  # 今日笔记
    ├── 2026-03-26.md
    └── ...
```

## 每日笔记结构

```markdown
# YYYY-MM-DD

## 今日待办

- [ ] 今日要完成的任务

## 长期待办

### 项目A

- [ ] 项目相关待办

### 项目B

- [ ] 项目相关待办

## 今日工作

-

## 会议/沟通

-

## 笔记

-

## 明日计划

-
```

## 核心规则

### 待办管理规则

- **今日待办**：今天需要完成的任务
- **长期待办**：没有明确日期的长期任务
- **唯一真相**：待办只存在于最新的每日笔记中

### 继承机制

新笔记创建时自动从前一天继承长期待办

#### 继承流程（自动）

创建新笔记时：

1. 读取前一天笔记
2. 复制"今日待办"中未完成的项目 → 今日笔记的"今日待办"
3. 复制"长期待办"全部内容 → 今日笔记的"长期待办"
4. 在前一天笔记末尾添加"（已继承到 YYYY-MM-DD）"标记

#### 示例

**3月26日笔记**（创建时）：

```markdown
# 2026-03-26

## 今日待办

- [ ] 和客户沟通

## 长期待办

### 山西项目

- [ ] 跟进项目申报

## 今日工作

- xxx
```

**3月27日新笔记创建时**：

- 从26日获取"今日待办"未完成项
- 从26日获取"长期待办"全部内容
- 26日笔记末尾添加"（长期待办已继承到 2026-03-27）"

**3月27日笔记**（创建后）：

```markdown
# 2026-03-27

## 今日待办

- [ ] 和客户沟通   ← 从26日继承

## 长期待办

### 山西项目

- [ ] 跟进项目申报  ← 从26日继承

## 今日工作

-

...

---

*长期待办已继承到 2026-03-27*  ← 26日笔记的标记
```

## 使用方式

### 手动创建今日笔记（带继承）

1. 确定今日日期（格式：YYYY-MM-DD）
2. 计算昨日日期
3. 读取昨日笔记内容
4. 提取待办信息，生成今日笔记
5. 更新昨日笔记（添加继承标记）

示例步骤：

```python
# 1. 列出 memory 目录确认日期
list_dir("memory")

# 2. 读取昨日笔记（如 2026-03-26）
read_file("memory/2026-03-26.md")

# 3. 创建今日笔记（2026-03-27），继承待办
write_file("memory/2026-03-27.md", content)

# 4. 更新昨日笔记，添加继承标记
edit_file("memory/2026-03-26.md",
    old_content="## 明日计划\n\n-",
    new_content="## 明日计划\n\n-\n\n---\n\n*长期待办已继承到 2026-03-27*")
```

### 定时自动执行（推荐）

配置 cron 任务每天自动创建笔记：

```bash
# 每天早上 7:00 自动创建今日笔记并继承待办
0 7 * * * 创建今日笔记并继承昨日待办
```

**执行流程**：

1. 计算今日和昨日日期
2. 检查昨日笔记是否存在
3. 如果存在，提取待办并继承
4. 创建今日笔记
5. 更新昨日笔记标记

### 追加内容

使用 `edit_file` 工具追加内容到当日笔记。

示例：

```python
edit_file("memory/2026-03-27.md",
    old_content="## 今日工作\n\n-",
    new_content="## 今日工作\n\n- 完成了xxx")
```

### 查看笔记

使用 `read_file` 工具读取指定日期的笔记。

示例：

```python
read_file("memory/2026-03-27.md")
```

### 列出最近笔记

使用 `list_dir` 工具列出 `memory/` 目录。

示例：

```python
list_dir("memory")
```

## 保留期限

- 每日笔记在 `memory/` 目录保留 7 天
- 超过 7 天的笔记会被归档到 `daily/YYYY/MM/` 目录
- 归档后可通过 `note_search` 工具搜索
- **注意**：归档后长期待办不会自动继承，需要时从归档中检索

## 与其他笔记的区别

| 类型       | 用途               | 位置                     |
| ---------- | ------------------ | ------------------------ |
| 每日笔记   | 记录当天事件、待办 | `memory/YYYY-MM-DD.md`   |
| MEMORY.md  | 长期信息（无待办） | `memory/MEMORY.md`       |
| 临时笔记   | 快速记录临时想法   | `pending/{主题}.md`       |
| 主题笔记   | 深度记录特定主题   | `topics/{主题}.md`       |
| 项目笔记   | 项目相关信息       | `projects/{项目名}.md`   |

## 注意事项

- 每日笔记应简洁，记录关键信息
- 使用标准的 Markdown 格式，便于后续检索
- **重要**：待办信息只放在每日笔记，MEMORY.md 不放待办
- 重要信息应转移到 MEMORY.md 或相关项目笔记
- 超过 7 天的笔记会被自动归档
- 建议配置 cron 任务每天自动执行，避免遗漏
