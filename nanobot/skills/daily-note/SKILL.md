# 每日笔记技能

## 功能

- 自动在 `memory/` 目录下创建每日笔记文件 `YYYY-MM-DD.md`
- 支持追加内容到当日笔记
- 支持查看最近几天的笔记
- 超过 7 天的笔记会被归档到 `daily/` 目录

## 目录位置

```
workspace/
└── memory/            # 每日笔记目录（最近 7 天）
    ├── MEMORY.md      # 长期记忆
    ├── HISTORY.md     # 历史日志
    ├── 2026-03-01.md
    ├── 2026-03-02.md
    └── ...
```

## 使用方式

### 创建今日笔记

使用 `write_file` 工具创建 `memory/YYYY-MM-DD.md` 文件。

示例：

```
write_file("memory/2026-03-02.md", "# 2026-03-02\n\n## 今日任务\n\n- [ ] 任务1\n- [ ] 任务2\n")
```

### 追加内容

使用 `edit_file` 工具追加内容到当日笔记。

示例：

```
edit_file("memory/2026-03-02.md", old_content="## 今日任务\n", new_content="## 今日任务\n- [ ] 新任务\n")
```

### 查看笔记

使用 `read_file` 工具读取指定日期的笔记。

示例：

```
read_file("memory/2026-03-02.md")
```

### 列出最近笔记

使用 `list_dir` 工具列出 `memory/` 目录。

示例：

```
list_dir("memory")
```

## 每日笔记模板建议

```markdown
# YYYY-MM-DD

## 今日任务

- [ ] 任务1
- [ ] 任务2

## 工作记录

-

## 会议/沟通

-

## 笔记

-

## 明日计划

-
```

## 保留期限

- 每日笔记在 `memory/` 目录保留 7 天
- 超过 7 天的笔记会被归档到 `daily/YYYY/MM/` 目录
- 归档后可通过 `note_search` 工具搜索

## 与其他笔记的区别

| 类型     | 用途               | 位置                   |
| -------- | ------------------ | ---------------------- |
| 每日笔记 | 记录当天事件、任务 | `memory/YYYY-MM-DD.md` |
| 临时笔记 | 快速记录临时想法   | `pending/{主题}.md`    |
| 主题笔记 | 深度记录特定主题   | `topics/{主题}.md`     |
| 项目笔记 | 项目相关信息       | `projects/{项目名}.md` |

## 注意事项

- 每日笔记应简洁，记录关键信息
- 使用标准的 Markdown 格式，便于后续检索
- 重要信息应转移到 MEMORY.md 或相关项目笔记
- 超过 7 天的笔记会被自动归档
