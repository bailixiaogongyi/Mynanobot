# Web 搜索技能

## 功能说明

使用博查搜索 API 在网络上搜索信息，适合国内网络环境。可以用来获取最新的新闻、技术资讯、产品信息等。

## 工具信息

- **工具名称**: `web_search`
- **功能**: 搜索网络，返回标题、URL、摘要和详细内容
- **适用场景**: 需要获取最新信息、了解时事新闻、查询技术文档等

## 使用方法

### 基本搜索

```python
web_search(query="最新人工智能发展趋势")
```

### 高级搜索

```python
web_search(
    query="2026年AI技术发展",
    count=10,  # 返回10个结果
    freshness="oneMonth",  # 只搜索一个月内的内容
    summary=True  # 包含详细摘要
)
```

### 时间范围选项

- `oneDay`: 一天内
- `oneWeek`: 一周内
- `oneMonth`: 一个月内
- `oneYear`: 一年内
- `noLimit`: 无时间限制（默认）

## 配置要求

使用此技能需要配置博查搜索 API key：

1. **环境变量方式**:
   - 设置 `BOCHA_API_KEY` 环境变量

2. **配置文件方式**:
   - 在 `~/.nanobot/config.json` 中添加：
   ```json
   {
     "tools": {
       "web": {
         "search": {
           "apiKey": "your_bocha_api_key"
         }
       }
     }
   }
   ```

## 最佳实践

### 1. 明确搜索意图

- 尽量使用具体的搜索词
- 包含时间范围限制以获取最新信息
- 明确搜索领域（如技术、新闻、产品等）

### 2. 处理搜索结果

- 分析搜索结果的相关性
- 查看来源可靠性
- 综合多个结果进行分析
- 引用时包含来源链接

### 3. 常见使用场景

#### 场景1: 新闻资讯查询
```python
web_search(query="2026年4月最新科技新闻", freshness="oneWeek")
```

#### 场景2: 技术文档查询
```python
web_search(query="Python 3.12 新特性", count=5, summary=True)
```

#### 场景3: 产品信息查询
```python
web_search(query="最新智能手机对比 2026", freshness="oneMonth")
```

## 错误处理

如果遇到以下错误：

1. **API key 未配置**:
   - 请按照配置要求设置 `BOCHA_API_KEY`

2. **搜索无结果**:
   - 尝试使用更具体的搜索词
   - 扩大时间范围
   - 检查网络连接

3. **API 调用失败**:
   - 检查 API key 是否有效
   - 检查网络连接
   - 稍后重试

## 注意事项

- 搜索结果可能存在时效性，请注意信息的发布时间
- 搜索结果仅供参考，需要自行验证信息的准确性
- 合理使用搜索功能，避免过度请求导致 API 限流
